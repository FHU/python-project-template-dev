bee_movie_analyzer module

Script to analyze a text file of ‘the bee movie’ movie script

bee_movie_analyzer.count_words(word_list)

The count_words function takes a word list and returns a dictionary with the word as key and the value is the word count for that word

bee_movie_analyzer.get_words(path)

The get_words function takes a file path and returns a filtered list of the words in the file. The words should all be lowercase It filters out all the punctuation except contractions (they count as their own words).

bee_movie_analyzer.nth_most_common_words(path, n)

The nth_most_common_words function takes the path to the script file and the word frequency rank and returns a list of words at that rank return an empty string if the rank is below 1 or if the rank does not exist

Parameters

  • path (str) – The path to the script file to analyze

  • n (int) – the rank of the word list returned

Returns

a list of all the words at rank n

Return type

List

bee_movie_analyzer.word_probability(path, word)

The word probability function takes a file path and a word and returns a float that is a decimal value of the probability that any single word in the script would be that word. If the word is not in the script, the probability is 0
